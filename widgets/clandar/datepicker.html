简介

日历选择是在Web中是最为常用的一个控件，用户对该控件的灵活性、实用性都有严格的要求，所以BUI中提供了快捷的方式来使用DatePicker控件。

DatePicker控件的配置十分灵活,我们可以通过各种参数配置来达到开发者想要的效果，这需要开发者对DatePicker控件的API有一定的了解。

日历选择的运用

自定义返回格式

在日历选择控件中，使用最广泛的需求是自定义返回格式：

示例说明：

  • yyyy 代表年
  • mm 代表月
  • dd 代表日
  • HH 代表时
  • MM 代表分
  • ss 代表秒

日期的选择范围

示例说明：

  • maxDate 选择日期范围中的最大日期
  • minDate 选择日期范围中的最小日期

动态日期选择范围

在大部分互联网应用中都需要日历选择控件能支持动态的日期范围限制，而BUI也很方便的实现了这个功能。

<script type="text/javascript">
BUI.use('bui/calendar', function (Calendar) {
    var datepicker = new Calendar.DatePicker({
        trigger:'#t1',
        autoRender : true
    });
    var datepicker2 = new Calendar.DatePicker({
        trigger:'#t2',
        autoRender : true
    });
    datepicker.on("selectedchange",function(e){
        datepicker2.set("minDate",e.value);
    });
    datepicker2.on("selectedchange",function(e){
        datepicker.set("maxDate",e.value);
    });
});
</script>

示例说明：

  • on 添加绑定时间
  • selectedchange 选中值改变事件
  • 把两个日期控件绑定了选中值改变的事件后，对他们各自的日期选择范围进行更改。即可达到动态的日期范围限制。
