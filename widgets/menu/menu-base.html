简介

菜单的应用范围很广，一般应用都会需要复杂的应用程序菜单。BUI中为了针对这一使用场景，提供了便捷的菜单功能。

菜单和列表

菜单是和列表一脉相承的，简单的菜单和列表的展示方式、选中方式几乎没有差别的，只是在实现上有所差异。菜单和列表两者都继承于同一个抽象接口BUI.Component.UIBase.ChildList，所以在使用上和列表几乎无异。更多的资料可以查看列表的介绍

菜单的操作

所有菜单都支持下面的操作：

  1. addItem( item ),addItemAt( item, index ) : 添加选项
  2. addItems( items ),addItemsAt( items,index ) ： 批量添加选项
  3. removeItem(item),removeItemAt(index) : 移除选项
  4. removeItems(items) : 批量移除选项
  5. updateItem( item ) ：更新选项
  6. setSelected( item ) ：设置选中项
  7. getSelected( ),getSelection( ) ：获取选中项

菜单的大部分操作都可以参看列表的操作，这里就不作一一介绍。
