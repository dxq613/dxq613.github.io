简介

在富客户端应用中浮层的应用场景有很多：

  • 鼠标移动到某个区域显示提示信息
  • 弹出框编辑内容
  • 显示警告信息等等

在接下来的章节里我们会详细的介绍Dialog、Message、Tooltip等控件，本章主要讲述浮层的特性和常用方法，首先来看一下Overlay实现的扩展（Mixin）

  • BUI.Component.UIBase.Position : 位置相关扩展，详细信息参看章节：位置和对齐
  • BUI.Component.UIBase.Align ： 控件对齐的扩展，详细信息参看章节：位置和对齐
  • BUI.Component.UIBase.Close ： 是否可以关闭控件，详细信息参看章节：关闭控件
  • BUI.Component.UIBase.AutoShow ：自动显示控件，详细信息参看章节：自动显示控件
  • BUI.Component.UIBase.AutoHide ：自动隐藏控件，详细信息参看章节：自动隐藏控件

显示隐藏的效果

overlay支持显示或者隐藏时添加动画效果可以通过effect属性控制：

  • effect :
    1. none : 没有任何效果的显示隐藏
    2. linear : 线性的显示隐藏
    3. fade : 渐变效果
    4. slide：下拉效果
  • duration : 执行时间(ms)
  • callback : 回调函数
